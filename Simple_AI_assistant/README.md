Asystent Sklepu Botanicznego 
 
**Status:** Funkcjonalny  
**Język:** Python 
**Model:** Qwen2.5-7B / OpenAI
 
Inteligentny asystent konwersacyjny dla sklepu botanicznego "Zielony Doom" z zaawansowanym systemem walidacji odpowiedzi.
 
**Kluczowe cechy:**
- 3-etapowy pipeline przetwarzania (klasyfikator → generator → walidator)
- Ochrona przed manipulacją i jailbreak attempts
- System retry logic z automatyczną regeneracją odpowiedzi
- Baza wiedzy (15 faktów) o roślinach i sklepie
- Interfejs Jupyter Notebook z ipywidgets
 
**Technologie:** OpenAI SDK, LM Studio, ipywidgets, Qwen2.5-7B

Kod jest do zastsowania głównie w Jupyter Notebook.
